# CRUD-operations

Created CRUD operations using express 
1) C - create product.
2) R - Read product.
3) U - Update data of existing product.
4) D - Delete product.

*---Modules---*
1.Express
2.Body-parser
3.Mongoose

Api's are created by uisng expressjs by creating a server and all the Api's are tested on Postman and data is stored in MongoDb.
